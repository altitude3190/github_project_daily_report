# github_project_daily_report

## install

```
$ cd github_project_daily_report

### make .env file
$ touch .env
$ vim .env

BASE_URL=xxx
GITHUB_TOKEN=yyy
OWNER=zzz

### use
### npm version >= 6.5.0
$ npm i
$ npx tsc --lib "es6"
$ node index.js -r hoge,fuga --since 2019-01-10

### detail usage
$ node index.js -h
```
